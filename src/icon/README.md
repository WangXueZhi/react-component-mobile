# API

| 属性        | 说明                                                     | 类型        | 默认值 |
|----------- |--------------------------------------------------------- | ---------- |-------|
| name       | icon名字，对应显示你要的icon图形                             | string     | ""    |
| className  | 添加class                                                 | string     | ""    |
| color      | 自定义icon字体颜色                                          | string     | ""    |
| size       | 自定义icon字体大小                                          | string     | ""    |
| clickEvent | 点击自身事件                                                | function   | null    |
